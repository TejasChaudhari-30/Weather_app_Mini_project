#  Mini Weather App

A simple weather application built using **Node.js**, **Express**, and **EJS** that fetches real-time weather data from a weather API and displays it in a clean UI.

---

##  Features

- Search weather by city name
- Displays temperature and weather conditions
- Uses server-side rendering with EJS
- Clean and simple UI
- Secure API key handling using environment variables

---

##  Tech Stack

- **Node.js**
- **Express.js**
- **EJS**
- **CSS**
- **Weather API**
